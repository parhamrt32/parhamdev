University portal

Repository north_east

The University Portal is a cutting-edge web application utilizing React, MongoDB, Express.js, Node.js, and JWT for streamlined university operations. It offers students and faculty an intuitive platform with secure access control. Students can manage course registrations, access records, and stay updated on events, while faculty efficiently handle courses and communication. The system includes a dynamic calendar, secure messaging, and role-based access. Admins have a robust dashboard for management. This project optimizes university processes, ensuring secure and efficient academic management
